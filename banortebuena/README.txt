Se trata de una web multifuncional que interactua con el usuario de forma nativa, ademas entrenamos un LLM que ayuda al aprendizaje continuo por medio de preguntaz y quizzes para el usuario, ademàs realizaba una simulaicon ficticia de tus finanzas a corto y largo plazo.

Para este proyecto se utilizo: HTML, CSS y JavaScript

Para desplegar el proyecto necesitas abrir tu explorador preferido, oprimir ctr + o,  abrir el archivo "index1.html" y apartir de ese paso el usuario puede navegar libremente entre pestañas
